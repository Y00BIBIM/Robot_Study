### 공부한 개념

- IP: 컴퓨터의 주소
- Port: 컴퓨터 안에서 프로그램을 구분하는 번호
- HTTP: 클라이언트와 서버가 요청/응답을 주고받는 규칙
- JSON: 데이터를 주고받기 위한 문자열 형식
- TCP: 안정적인 연결 기반 통신
- UDP: 빠른 전송 중심 통신
