# Robot Intern Study

## 목표

Unity + ROS + FastAPI + DB 기반 가상 로봇 관제 미니 시스템을 만든다.

## 6월 21일 학습 내용

오늘은 Python 프로젝트 개발 환경을 구성했다.

### 공부한 내용

- Python 가상환경 venv 생성
- pip 패키지 설치
- requirements.txt 생성
- Python 프로젝트 폴더 구조 생성
- RobotState 클래스 작성
- CLI 기반 로봇 명령 시뮬레이터 작성
