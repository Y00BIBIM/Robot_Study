# Day 01 - ROS 1 Basic Concepts

## 오늘 배운 것
- ROS Master
- Node
- Topic
- Message
- Service

## ROS Master
ROS 노드들이 서로를 찾을 수 있도록 도와주는 중심 역할을 한다.

## Node
ROS에서 실행되는 하나의 프로그램 단위이다.

## Topic
Node들이 데이터를 주고받는 통신 통로이다.

## Message
Topic을 통해 전달되는 데이터 형식이다.

## Service
요청과 응답 방식의 통신이다.

## 오늘 사용한 명령어
- roscore
- rosnode list
- rosversion -d
